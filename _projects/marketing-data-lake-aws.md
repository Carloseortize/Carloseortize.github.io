    Este repositorio contiene un proyecto profesional de Data Analytics que implementa un Data Lake en Amazon Web Services (AWS) para analizar el rendimiento de campañas de marketing. El objetivo es demostrar un flujo de trabajo completo en la nube, desde el almacenamiento de datos hasta la obtención de insights de negocio, utilizando un dataset público de inversión publicitaria en TV, radio y prensa.    **Herramientas:** *AWS, SQL, Data Lake, Python*

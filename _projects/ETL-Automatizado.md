    Este proyecto implementa un pipeline ETL automatizado para analizar datos de viajes de taxi utilizando tecnologías cloud. El pipeline procesa datos de movilidad urbana utilizando Python y Google Cloud y automatiza el flujo de datos con Apache Airflow.
    **Herramientas:** *Python, Pandas, Google Cloud Storage, BigQuery, Apache Airflow, Matplotlib, SQL*
